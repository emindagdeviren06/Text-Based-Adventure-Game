#Text Based Adventure Game

A command-line text based adventure game developed in Python.
Players navigate through the story by making their own choices, in which affect the outcome of the game.

##Features
-Interactive text-based gameplay
-Branching story paths throughout
-Player decision-making system
-Input handling and game logic

##Languages 
-Python

##How to Run
The instructions are explained to the fullest detailed when the game is run.

##What I Learned
-Structuring programs using functions and modules (python)
-Handling user input and control flow
-Designing interactive systems
